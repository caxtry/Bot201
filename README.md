# BOT SETUP

## Installation ##
```bash
npm install
```

## Node
* Node version: `v16.20.0`

## Token
* Change your token and more in --> `.env`

## Start Bot
type:
```bash
node .
```
in your terminal

or
```bash
npm start index.js
```

## Usage in discord
```bash
//help
```
* // is the main prefix. You can change it in --> `botconfig/config.json`
